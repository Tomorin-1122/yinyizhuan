### 任务 P1-4：BibTeX 嵌套花括号解析

**位置**：`src/lib/bibtex-parser.ts:14`

**现状**：
```ts
const fieldRegex = /([a-zA-Z]+)\s*=\s*[\{"]?([\s\S]*?)[\}"]?[\s\n]*,/g;
```
无法处理 `title = {A {Deeply} Nested {Example}}`（标准 BibTeX 常见），会截断到第一个 `}`。

**改法**：写一个状态机解析器替代正则：
```ts
function parseBibFields(text: string): Record<string, string> {
  const fields: Record<string, string> = {}
  let i = 0
  while (i < text.length) {
    // 跳过空白和逗号
    while (i < text.length && /[\s,]/.test(text[i])) i++
    if (i >= text.length) break
    // 读字段名
    let name = ''
    while (i < text.length && /[a-zA-Z]/.test(text[i])) name += text[i++]
    // 跳过 = 和空白
    while (i < text.length && /[\s=]/.test(text[i])) i++
    // 读值（支持 { } 嵌套 和 " "）
    let value = ''
    const quote = text[i]
    if (quote === '{') {
      let depth = 1; i++
      while (i < text.length && depth > 0) {
        if (text[i] === '{') depth++
        else if (text[i] === '}') depth--
        if (depth > 0) value += text[i]
        i++
      }
    } else if (quote === '"') {
      i++
      while (i < text.length && text[i] !== '"') value += text[i++]
      i++
    } else {
      while (i < text.length && !/[\s,}]/.test(text[i])) value += text[i++]
    }
    if (name) fields[name.toLowerCase()] = value.trim()
  }
  return fields
}
```

**验收**：
- [ ] 用包含 `{A {B} C}` 的 .bib 测试，title 完整保留为 `A {B} C`（或剥掉内层括号，二选一，与 BibTeX 标准一致）
- [ ] 原有的简单 .bib 仍能正确解析

   export interface FieldError {
     field: string
     message: string

   export function validateCitation(c: Citation): FieldError[] {
     const errors: FieldError[] = []
     if (!c.title.trim()) {
       errors.push({ field: 'title', message: '标题不能为空' })
     }
     if (c.authors.length === 0 || c.authors.every(a => !a.name.trim())) {
       // 古籍/日记等特殊类型可不填作者，按类型判断
       if (!['ancient', 'classic', 'diary'].includes(c.type)) {

       errors.push({ field: 'publishYear', message: '年份格式应为 YYYY 或 YYYY-YYYY' })

       errors.push({ field: 'pages', message: '页码格式异常' })
     }
     return errors
   }

   - 若 P1-1 拆分时已建 `ConfirmDialog` 通用组件则复用
   - 若没有，用项目已有的 Toast/Modal 风格写一个：
   ```tsx
   const [showResetConfirm, setShowResetConfirm] = useState(false)

   const handleReset = () => setShowResetConfirm(true)
   const confirmReset = () => {
     setCitation(defaultCitation())
     setShowResetConfirm(false)
   }
   ```
   ```tsx
   {showResetConfirm && (
     <div className="fixed inset-0 bg-black/30 flex items-center justify-center z-50">
       <div className="bg-white rounded-lg p-6 max-w-sm">

         <div className="flex gap-3 justify-end">
           <button onClick={() => setShowResetConfirm(false)}>取消</button>
           <button onClick={confirmReset} className="bg-red-500 text-white px-4 py-1 rounded">确认清空</button>
         </div>
       </div>
     </div>
   )}
   ```

3. **智能跳过**：表单未填任何内容时（title 为空且 authors 只有空行），直接清空不弹窗
   ```tsx
   const handleReset = () => {
     const isEmpty = !citation.title.trim() && citation.authors.every(a => !a.name.trim())
     if (isEmpty) {
       setCitation(defaultCitation())
       return
     }
     setShowResetConfirm(true)
   }
   ```
