# 项目最新状态记录

## 更新时间
2026年6月21日

## 项目概况
**引易转** (YinYiZhuan) - 中国历史学者学术引用格式转换工具

### 当前版本状态
- 最近更新：第二批功能优化方案全部执行完成（6 项需求 + 3 项修复）
- 支持 12 种文献类型，古籍支持 7 种子类型
- 部署在 Vercel：https://yinyizhuan.vercel.app

### 核心功能
1. **多格式转换**：《历史研究》、GB/T 7714-2015、APA 第7版
2. **多输入方式**：手动填写、文本解析、URL/DOI/ISBN 导入、文件上传（含拖拽）
3. **批量处理**：支持多条引用同时转换
4. **本地存储**：浏览器本地存储转换历史（最多 500 条）
5. **Word 导出**：支持导出为 Word 文档
6. **输入校验**：仅期刊强制要求标题，其他类型可不填
7. **重置确认**：清空表单前二次确认
8. **再次引证**：历史记录页支持一键复制省略出版信息的引证格式
9. **期刊页码切换**：转换结果区可切换显示/隐藏页码

### 技术栈
- React 18 + TypeScript + Vite 5
- Tailwind CSS
- React Router 7 路由
- opencc-js 简繁转换
- docx 库 Word 导出
- vitest 测试框架（50 个用例）

### 项目结构
```
src/
├── lib/                    # 核心逻辑（无 React 依赖）
│   ├── types.ts           # 类型定义（含 ancientSubType）
│   ├── parser.ts          # 文本解析（含 MLA 解析器）
│   ├── formatters/        # 格式化器
│   │   ├── lsyj.ts       # 《历史研究》格式（12 种文献类型，export processBookTitleMarks）
│   │   ├── gbt7714.ts    # 国标格式
│   │   ├── apa.ts        # APA 格式
│   │   └── author-utils.ts # 共享作者格式化原语
│   ├── utils.ts           # 工具函数（generateId、detectLanguage 等）
│   ├── validate.ts        # 输入校验（仅期刊强制标题）
│   ├── use-access.ts      # 访问状态 hook
│   ├── storage.ts         # 本地存储管理
│   ├── metadata-fetcher.ts # URL/DOI/ISBN 元数据抓取
│   ├── bibtex-parser.ts   # BibTeX 文件解析（支持嵌套花括号）
│   ├── ris-parser.ts      # RIS 文件解析
│   ├── export-word.ts     # Word 文档导出
│   └── access.ts          # 访问控制（SHA-256 哈希校验）
├── pages/                  # 页面组件
│   ├── ConvertPage.tsx    # 核心转换页（~500 行）
│   ├── HistoryPage.tsx    # 历史记录页（含再次引证按钮）
│   ├── HomePage.tsx       # 首页
│   ├── AboutPage.tsx      # 关于页
│   └── InvitePage.tsx     # 邀请码输入页
└── components/             # 共享组件
    ├── Layout.tsx          # 布局组件
    ├── Icons.tsx           # 图标组件（含 IconRepeat、IconChevronDown）
    ├── ManualForm.tsx      # 手动填写表单（古籍子类型下拉）
    ├── InviteModal.tsx     # 邀请码弹窗
    ├── FileUploadArea.tsx  # 文件上传区域
    ├── AddToGroupButton.tsx # 标签组按钮
    └── AccessQuotaDisplay.tsx # 配额显示
```

### 测试

```bash
npm run build   # TypeScript + Vite 生产构建
npm run lint    # ESLint 检查
```

测试覆盖：

- `formatters.test.ts`：20 个 snapshot 用例（12 种文献类型 × 3 种格式）
- `bibtex-parser.test.ts`：5 个用例（含嵌套花括号）
- `publisher-places.test.ts`：16 个用例

### 开发注意事项
1. **中文字符处理**：TSX/JSX/HTML 中必须使用真实中文字符，禁止 Unicode 转义
2. **Vercel 部署**：必须先 git push 再 vercel deploy，否则服务端拉取旧代码
3. **Layout.tsx CRLF 换行**：该文件含 CRLF，edit 工具无法匹配，必须用 write 整体重写
