# 项目最新状态记录

## 更新时间
2026年6月20日

## 项目概况
**引易转** (YinYiZhuan) - 中国历史学者学术引用格式转换工具

### 当前版本状态
- 最近更新：代码改进方案全部执行完成（P0~P3 共 12 项），新增用户反馈修正
- 支持 12 种文献类型，包括最新的古籍格式优化
- 部署在 Vercel：https://yinyizhuan.vercel.app

### 核心功能
1. **多格式转换**：《历史研究》、GB/T 7714-2015、APA 第7版
2. **多输入方式**：手动填写、文本解析、URL/DOI/ISBN 导入、文件上传（含拖拽）
3. **批量处理**：支持多条引用同时转换
4. **本地存储**：浏览器本地存储转换历史（最多 500 条）
5. **Word 导出**：支持导出为 Word 文档
6. **输入校验**：标题/作者为空时红框提示，点转换时拦截
7. **重置确认**：清空表单前二次确认

### 技术栈
- React 18 + TypeScript + Vite 5
- Tail CSS
- React Router 7 路由
- opencc-js 简繁转换
- docx 库 Word 导出
- vitest 测试框架（34 个用例）

### 项目结构
```

npm run lint    # ESLint 检查
```

测试覆盖：

- `formatters.test.ts`：20 个 snapshot 用例（12 种文献类型 × 3 种格式）
- `bibtex-parser.test.ts`：5 个用例（含嵌套花括号）

### 开发注意事项
1. **中文字符处理**：TSX/JSX/HTML 中必须使用真实中文字符，禁止 Unicode 转义
2. **Vercel 部署**：必须先 git push 再 vercel deploy，否则服务端拉取旧代码
3. **Layout.tsx CRLF 换行**：该文件含 CRLF，edit 工具无法匹配，必须用 write 整体重写
4. **GitHub 代理**：推送需要走 Clash 代理（端口 17890），加 `-c http.proxy=http://127.0.0.1:17890 -c http.sslVerify=false`
